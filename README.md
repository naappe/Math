# Mobile Screen Recorder

This project is the original single-file mobile screen recorder organized into separate files.

## Files

- `index.html` — page structure
- `styles.css` — the original embedded CSS
- `app.js` — the original embedded JavaScript
- `original-one-file.html` — untouched backup of the uploaded file

## GitHub Pages

Upload all files to the same folder. Open `index.html`.

No layout, CSS rule, element ID, event handler, or recording logic was intentionally changed.
